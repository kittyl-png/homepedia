# P0.3 — 全局样式集成指南

## 产出文件

| 文件 | 作用 |
|------|------|
| `app/fonts.ts` | 字体加载配置（Montserrat + Outfit） |
| `app/globals.css` | CSS 变量 + Tailwind 4 theme + 全局样式 |

---

## 集成到 Layout（P0.4 需做）

在根布局 `app/[locale]/layout.tsx`（或 `app/layout.tsx`）中注入字体 CSS 变量：

```tsx
import { montserrat, outfit } from "../fonts";
import "../globals.css";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html className={`${montserrat.variable} ${outfit.variable}`}>
      <body>{children}</body>
    </html>
  );
}
```

`montserrat.variable` 会注入 `--font-montserrat`，`outfit.variable` 注入 `--font-outfit`。
`globals.css` 中的 `@theme inline` 通过这两个变量定义 `--font-heading` 和 `--font-sans`。

## 设计 Token 速查

### 颜色（Tailwind 类名）

| 设计系统名称 | Tailwind 类 | 色值 | 用途 |
|-------------|------------|------|------|
| 品牌深蓝 | `text-brand-blue` / `bg-brand-blue` | `#004aad` | 标题、Logo、导航、主文字色 |
| 品牌青色 | `text-brand-cyan` / `bg-brand-cyan` | `#5de0e6` | 小标签、icon 强调、hover |
| 行动金色 | `text-brand-gold` / `bg-brand-gold` | `#ffbd59` | CTA 按钮、重要标记 |
| 金色 hover | `bg-brand-gold-hover` | `#ffaa2b` | CTA hover |
| 暖白背景 | `bg-warm-white` | `#f8f8f3` | 页面背景、交替 section |
| 辅助灰 | `text-slate-400` / `text-slate-500` | Tailwind 内置 | 正文、次要描述 |

### 字体（Tailwind 类名）

| 用途 | Tailwind 类 | 字体 |
|------|------------|------|
| 标题 h1/h2/h3 | `font-heading` | Montserrat |
| 正文 / UI | `font-sans`（默认） | Outfit |

### 阴影（Tailwind 类名）

| 场景 | Tailwind 类 |
|------|------------|
| 默认卡片 | `shadow-card` |
| Hover 卡片 | `shadow-card-hover` |
| 重要容器 | `shadow-elevated` |
| CTA 按钮 | `shadow-cta` |
| Logo 背景 | `shadow-logo` |

### CSS 变量（可直接在内联样式或自定义 CSS 中使用）

```css
var(--brand-deep-blue)
var(--brand-cyan)
var(--brand-gold)
var(--brand-warm-white)
var(--gradient-brand)    /* 品牌渐变 */
```

## 组件样式速查（从设计系统提取）

### Section 小标签
```tsx
<span className="text-xs font-bold text-brand-cyan uppercase tracking-[0.3em] mb-4 block">
  Our Expertise
</span>
```

### CTA 按钮（主要 — 金色）
```tsx
<button className="bg-brand-gold hover:bg-brand-gold-hover text-brand-blue font-bold py-4 px-10 rounded-xl shadow-cta active:scale-95 transition-all">
  Start with AI Assistant
</button>
```

### CTA 按钮（次要 — 深蓝）
```tsx
<button className="bg-brand-blue hover:bg-brand-cyan text-white font-bold py-5 px-10 rounded-xl transition-colors">
  Contact Us
</button>
```

### 卡片
```tsx
<div className="bg-white p-8 rounded-2xl shadow-card hover:shadow-card-hover transition-shadow duration-300 border border-slate-100">
  ...
</div>
```

### 表单输入框
```tsx
<input className="bg-warm-white border border-slate-200 p-4 rounded-xl outline-none focus:border-brand-blue transition-colors" />
```

## 注意事项

- **不使用纯黑** `#000000` — 所有深色文字用 `text-brand-blue`（深蓝 `#004aad`）
- **不使用纯灰阴影** — 阴影都带品牌色色调（已在 `@theme` 中配置）
- **不使用 dark mode** — 设计系统没有暗色方案
- **中文不加载 web font** — `system-ui` 回退即可，节省加载时间
- **字体只加载必要字重** — Montserrat 600/700, Outfit 300/400/500/600，不加载 italic
